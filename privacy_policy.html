Privacy Policy

Last Updated: March 2025

This Privacy Policy describes how eBay Postcard Lister ("the Application", "we", "our") handles information when you use our desktop application to create eBay listings.

Information We Collect

The Application collects and processes the following information locally on your computer:

  • eBay API Credentials: Your eBay App ID, Cert ID, Dev ID, and OAuth tokens required to interact with the eBay API
  • OpenAI API Key: Your API key for generating listing descriptions
  • Images: Postcard images you select for listing
  • Listing Data: Titles, descriptions, prices, and inventory information you create

How We Use Information

All information is used solely for the purpose of:

  • Authenticating with eBay's API to create and manage listings
  • Sending images to OpenAI's API to generate listing descriptions
  • Storing your inventory and listing history locally

Data Storage

All data is stored locally on your computer in the application directory. We do not operate servers or cloud storage. Specifically:

  • Configuration is stored in config.json
  • OAuth tokens are stored in ebay_tokens.json
  • Inventory data is stored in inventory.json
  • Listing history is stored in listings.json

Third-Party Services

The Application interacts with the following third-party services:

  • eBay API: To create and manage your eBay listings. Subject to eBay's Privacy Policy
  • OpenAI API: To analyze images and generate listing content. Subject to OpenAI's Privacy Policy

Data Sharing

We do not sell, trade, or share your personal information with third parties. Data is only transmitted to:

  • eBay's servers when creating/managing listings (as initiated by you)
  • OpenAI's servers when generating descriptions (as initiated by you)

Data Security

Your API credentials and tokens are stored locally on your computer. We recommend:

  • Keeping your computer secure with appropriate access controls
  • Not sharing your configuration files with others
  • Revoking eBay OAuth tokens if you suspect unauthorized access

Your Rights

Since all data is stored locally on your computer, you have full control over it. You can:

  • View all stored data by opening the JSON files in the application directory
  • Delete any or all data by removing the configuration files
  • Revoke eBay access by deleting tokens or revoking access in your eBay account settings

Children's Privacy

This Application is not intended for use by children under the age of 18. We do not knowingly collect information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this page.

Contact

If you have questions about this Privacy Policy or the Application's data practices, please contact the application developer.
